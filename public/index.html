welcome to ubiquo!

the CMS framework for pragmatic developers.

  • Configure your database using the config file config/database.yml

    You can follow detailed instructions provided here.
  • Run rake ubiquo:db:init to initialize the database.

    If you need more info about rake check here.

  • Restart your server script/server and go to http://localhost:3000/ubiquo

    A superadmin user will be created upon login using the provided login and password.
